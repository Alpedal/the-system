# Variant: Living Machine

## Design stance
Organisk, levande — gränssnittet känns som en varelse, inte ett verktyg. Partiklar i bakgrunden, pulserande agent-ringar, chatt som andas. Som om Igris är en levande AI-entitet.

## Key choices
- Layout: Centrerad chatt (max 720px), agent-ringar i topbar, GPU-strip i botten
- Typography: Inter för body, JetBrains Mono för data
- Color: Djupare bakgrund (#06060c), cyan partiklar, glow-effekter
- Interaction: Ringar animeras med pulserande bågar, bakgrund lever med partikel-nätverk, chatt-tonas in

## Trade-offs
- Strong at: Känslomässig koppling, unik estetik, levande känsla
- Weak at: Mindre informationsdensitet, agent-ringar visar ingen detaljdata, inga siffror synliga utan klick

## Best for
Första intryck, demos, "wow"-faktor. Användare som vill känna att de pratar med en levande AI snarare än ett verktyg.
